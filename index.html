Help Information

For reference, the SVG area is 600 X 600 pixels. Use the input fields to adjust the x y coordinates of the start, and end points (0 - 600).

Similarly, you can adjust the location of the control point (which can be off the display screen). The curve will update dynamically as you change the values.

You can use the save button to copy the displayed data (d) parameter for the path into the clip board. Then you can paste it into your application.
